Aim

Our aim about his project is to classify the sentiment of companies based on verified user's tweets as well as articles published by reputable sources.

For example, a tweet would say Amazon is a buy, you must invest now and these libraries would classify it as negative , positive or neutral sentiment.

Problem Statement

Compute and display statistics.
Visualize the data and its outcome by plotting graphs.
Perform variable transformations and also describe correlations for the variables. Build a model and predict results. Perform visual analytics using any Tableau.Create a website to showcase the entire work done.

In detail get more info in Objective section.

1.Cleaning of Tweets: Removing the stop words.
